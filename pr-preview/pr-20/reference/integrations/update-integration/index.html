Create or update an integration

This endpoint allows for creating or updating an integration within Connxio. Users can programmatically configure the integration properties using the same properties available in the Connxio Web Portal UI. This endpoint supports creating new integrations or updating existing ones.

URL: https://api.connxio.no/integrations/:integrationConfigId

Method: PUT

Route Parameters:

KeyTypeDescription
integrationConfigIdstringThe ID of the integration.

Request body:

KeyTypeOptionalDescription
idstringnoThe ID of the integration.
namestringnoThe name of the integration.
descriptionstringnoA description of the integration.
senderstringnoThe name of system that sends the data.
receiverstringnoThe name of receiving system.
enabledbooleannoWhether the integration is enabled.
inboundConnectionobjectnoConfiguration of the inbound connection.
subIntegrationsarraynoConfiguration of the subintegrations.
