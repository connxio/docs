# Messaging API

The Messaging API in Connxio provides users with a programmatic interface to interact with the messaging capabilities of Connxio. It allows users to push messages to Connxio for further processing and integration, providing flexibility and control over data ingestion and exchange within their systems.

Check out the operations listed below to learn more about the Messaging API and how to use it.
- [Send a single message](/reference/messages/send-single-message)
- [Send a batch of messages](/reference/messages/send-batch)
- [Send messages via Azure Event Grid](/reference/messages/send-event-grid)
