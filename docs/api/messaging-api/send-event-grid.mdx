# Send an Event Grid message

Use this endpoint when using Azure Event Grid to messages to Connxio.


**URL:** `https://api.connxio.no/messages/:integrationConfigId/eventgrid`

**Method:** <span class="method post">POST</span>

**Route Parameters:**

| Key        | Type   | Description |
| ---------- | ------ | ----------- |
| integrationConfigId | string | The ID of the integration configuration to use when sending the message. |

**Query Parameters:**

| Key        | Type   | Optional | Description |
| ---------- | ------ | -------- | ----------- |
| Connxio-Api-Key | string | no | Event Grid does currently support adding custom headers, so you need to add the API key as a query parameter. |

**Request body:** 

| Key        | Type   | Optional |
| ---------- | ------ | -------- |
| topic | string | no |
| subject | string | yes |
| eventType | string | no |
| eventTime | ISO 8601 date | yes |
| id | string | yes |
| data | string | no |
| dataVersion | string | yes |
| metadataVersion | string | yes |

### **Example request**

```json
// POST https://api.connxio.no/messages/6cd585f1-e408-4f00-9253-76b717ab12de

// REQUEST BODY
[
  {
    "topic": "string",
    "subject": "string",
    "eventType": "string",
    "eventTime": "2023-03-31T19:29:12.612Z",
    "id": "string",
    "data": "string",
    "dataVersion": "string",
    "metadataVersion": "string"
  }
]

// RESPONSE
200 OK
```
